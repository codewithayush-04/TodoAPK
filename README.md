## ⚙️ Setup (Supabase)

1. Create a project on **Supabase**
2. Open **SQL Editor**
3. Run:npm run  dev

4. Enable authentication:
- Go to **Authentication → Providers**
- Enable **Email**

## 💻 Local Development

### 1. Clone the repository
```bash
git clone https://github.com/codewithayush-04/task-grid.git
cd task-grid
cp .env.example .env

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
npm install
npm run dev
npm run build
npm run preview
```
